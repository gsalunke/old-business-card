Add business card to your existing fastn site

This step-by-step guide will help you to integrate your personalised business card into your existing fastn repository on GitHub. This entirely runs on the browser. Simply follow along the steps to integrate.

Step 1: Sign In to your GitHub account

  • Sign In to your GitHub account to access your fastn repository

Step 2: Identify the existing fastn repository on GitHub

  • Select the existing fastn repository on GitHub to which you want to integrate the business card
You can find the fastn repository under Repositories on GitHub

Step 3: Navigate to the FASTN.ftd document

  • Navigate to and open the FASTN.ftd file of the selected project
  • Click on the edit icon (as shown below) to make the file editable

Step 4: Add below line inside your FASTN.ftd document

  • Copy the following code
null
-- fastn.dependency: fastn-community.github.io/business-card

-- fastn.dependency: fastn-community.github.io/business-card
Lang:
ftd
  • Paste the code in the FASTN.ftd file along with other dependencies, if any
  • Click on the button Commit Changes...
A pop-up will appear.
You can check the Commit message and other details on the commit status bar
  • Refresh the browser to check if the commit workflow ran successfully. You will see a green tick

Step 5: Navigate to the document

  • Open the document to which you want to integrate the business card and display in the browser

    For example: In my case, I want to integrate business card in index.ftd file
  • Click on the Edit icon to make the file editable

Step 6: Add below line inside your ftd file

  • Copy the following import line
null
-- import: fastn-community.github.io/business-card

-- import: fastn-community.github.io/business-card
Lang:
ftd
  • Paste the import line at the top of the ftd file.

If this is the only import line, then it goes at line 1 else it can be placed below other already imported lines.

To read why to import at the top, Click Here

  • Paste the code where you want your business card to position

You can position the card at the start, in between the document or at the end.

In my case, I have added in between of the document after the third paragraph.

  • Commit the changes

  • Go to Actions tab and wait for the pages build and deployment to successfully complete

  • Once done, open the URL which you can find either through the Settings > Pages or if you have already added it in the About section in the first tab(Code tab)

You will be able to see the default business card in your fastn repository.
